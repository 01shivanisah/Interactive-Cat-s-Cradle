# Interactive-Cat-s-Cradle
A real-time hand tracking web application built using MediaPipe Hands and HTML5 Canvas.

## Features
- Real-time hand detection
- Tracks up to 2 hands
- Glowing hand skeleton visualization
- Dynamic fingertip string connections
- FPS monitoring
- Responsive fullscreen interface

## Technologies Used
- HTML5
- CSS3
- JavaScript
- MediaPipe Hands
- Canvas API

## How to Run
1. Download the project.
2. Open `index.html` in a browser.
3. Allow camera permissions.
4. Start interacting with your hands.
